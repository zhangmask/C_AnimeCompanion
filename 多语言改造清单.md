# 多语言支持改造清单（中文 → 英文）

## 改造目标
设置中新增"语言"下拉菜单，可选 `中文 / English`。选择英文后，所有前端 UI 字符串切换为英文显示。

## 技术方案
1. 新增 `AppLanguage` 枚举（`ZH / EN`）与持久化（DataStore）
2. 新增 `Strings.kt` 字符串表，按 key 提供 zh/en 两套文案
3. 在 Composable 中通过 `LocalLanguage` CompositionLocal 读取当前语言，从字符串表取值
4. `LanguageSettingsScreen` 改为下拉菜单（替换"即将上线"占位）
5. 所有 UI 中的中文硬编码字符串替换为 `Strings.get(key)` 调用
